Sistema de monitorización ambiental basado en hardware open-source alimentado mediante energía fotovoltaica Open Source

El objetivo del proyecto es crear una estación autosuficiente de monitorización que permita visualizar en tiempo real las condiciones ambientales en las que se encuentra un sistema concreto, así como recopilar datos de otros sistemas adyacentes; por ejemplo, instalaciones eléctricas, hidráulicas, etc.

Para ello, se utilizará un sensor transmisor en RF encargado de capturar todos estos datos y enviarlos a un módulo diseñado específicamente para Raspberry Pi, un ordenador de propósito general de bajo presupuesto. Este manejará y dará formato los datos recibidos y los almacenará en un servidor, local o en la Red, para su posterior interpretación y visualización.

Todo el conjunto se alimentará con energías renovables, en este caso fotovoltaica, para hacerlo totalmente independiente.

Como complemento se pueden añadir más módulos a los especificados para incrementar las funcionalidades de la monitorización y así, adaptarlo a las necesidades requeridas.
